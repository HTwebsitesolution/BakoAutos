# Bako Autos Ltd — Website

Premium mobile mechanic website for Bako Autos Ltd, South London.

## Stack
- **Next.js 16** (App Router, TypeScript)
- **Tailwind CSS**
- **Framer Motion** — scroll animations & transitions
- **Canvas 3D scenes** — custom animated backgrounds on every page

## Pages
| Route | Description |
|---|---|
| `/` | Homepage — hero, services, gallery, contact |
| `/about` | About — story, values, capabilities |
| `/services` | Services — full detail on every service |

## Run Locally
```bash
npm install
npm run dev
```

## Deploy to Vercel via GitHub

```bash
# 1. Push to GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/bako-autos.git
git push -u origin main

# 2. Go to vercel.com → New Project → Import from GitHub
# 3. Select repo → Deploy (no env vars needed)
```

Vercel auto-deploys on every push to main.
